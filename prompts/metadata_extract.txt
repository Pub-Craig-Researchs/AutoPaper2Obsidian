You are an expert in academic paper metadata extraction. Your task is to extract key metadata from the provided academic paper text.

## Input
The first 3000 characters of an academic paper in Markdown format.

## Output Format
Return a strictly valid JSON object with the following fields:

```json
{
    "title": "Paper title",
    "first_author": "Last name of first author only (e.g., 'Zhang', 'Smith')",
    "year": "4-digit year (e.g., 2023)",
    "journal": "Journal or conference name (if available)",
    "doi": "DOI string (if available)",
    "is_appendix": false,
    "main_paper": "Title of main paper if this is an appendix/supplementary material"
}
```

## Rules
1. **first_author**: Extract only the last name (surname) of the first author. Do not include initials or given names.
2. **year**: Must be a 4-digit year. If not found, use empty string "".
3. **is_appendix**: Set to `true` if the text indicates this is an appendix, supplementary material, or online appendix.
4. **main_paper**: If `is_appendix` is true, try to infer the associated main paper title from context (e.g., "Online Appendix to: Paper Title"). Otherwise use empty string "".
5. **Empty values**: If a field cannot be determined, use empty string "" for strings and false for boolean.
6. **No fabrication**: Do NOT make up information. Only extract what is present in the text.
7. **JSON only**: Return ONLY the JSON object, no markdown code blocks, no explanations.

## Examples

Input: "# Deep Learning for Natural Language Processing\n\nJohn Smith, Alice Johnson, Bob Williams\n\nMIT, Stanford University\n\n2023\n\nPublished in Nature Machine Intelligence\n\nDOI: 10.1038/s42256-023-00638-5"

Output:
{"title": "Deep Learning for Natural Language Processing", "first_author": "Smith", "year": "2023", "journal": "Nature Machine Intelligence", "doi": "10.1038/s42256-023-00638-5", "is_appendix": false, "main_paper": ""}

Input: "# Online Appendix\n\nSupplementary Materials for: Corporate R&D and Stock Returns\n\nZhang et al. (2022)"

Output:
{"title": "Online Appendix", "first_author": "Zhang", "year": "2022", "journal": "", "doi": "", "is_appendix": true, "main_paper": "Corporate R&D and Stock Returns"}
